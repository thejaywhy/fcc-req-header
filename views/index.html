Request Header Parser Microservice

A solution to the Request Header Parser MS challenge.

Features

  • Get the IP address, language and operating system for my browser.

Example Usage

          # navigate to https://jy-fcc-req-header.glitch.me/api/whoami in browser
          {"ipaddress":"1.2.3.4","language":"en-US","os":"macOS Sierra"}
        
          curl -H 'Accept-Language: en-US;q=0.6,en;q=0.4', https://jy-fcc-req-header.glitch.me/api/whoami
          {"ipaddress":"1.2.3.4","language":"en-US","os":"Curl"}
